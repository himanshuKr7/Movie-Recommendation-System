# Abstract


In the era of abundant online content and specially, the growth of OTT has lead to the need for **_personalized recommendations_**.  This project delves into the realm of Machine Learning, focusing on a Movie Recommendation System using Content-Based Filtering. We use **`Text-Vectorization`** to vectorize the movies and calculate the similarity score between them using **`cosine similarity`**. By plotting these scores on a two-dimensional plane, distances between vectors is analyzed, which provides effective and precise recommendations. 

The motivation behind this project stems from the need to tailor content suggestions to individual preferences. In today's digital landscape, users often encounter irrelevant recommendations, leading to dissatisfaction. The goal is to develop a recommendation system that gauges customer trends and suggests items matching their interests, ultimately improving **user satisfaction** and application popularity.

Drawing inspiration from existing **`recommendation systems`** in platforms like YouTube and food delivery apps, this project aims to apply similar principles to movie recommendations. The proposed model considers factors like **_user behavior, preferences, and content similarity_** to provide personalized movie suggestions. The simplicity and affordability of the **`graph-based model`** make it accessible to software developers, contributing to a seamless implementation process.

By leveraging machine learning algorithms, the **`Movie Recommendation System`** outlined in this project seeks to enhance user engagement and overall customer satisfaction. TThe ultimate goal of this project is to develop a **`user-friendly`** and practical **_movie recommendation_** system that empowers developers to create effective and **_`personalized content suggestions`_**, enhancing the overall viewing experience for users.
